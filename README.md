# ADA Lab Experiment 2

## Algorithmic Strategies Mini Project

### 📌 Description

This project demonstrates the implementation and analysis of various algorithmic strategies including Divide and Conquer, Greedy Algorithms, Dynamic Programming, and the Travelling Salesman Problem (TSP). The goal is to compare their efficiency and understand their real-world applications.

---

### ⚙️ Algorithms Implemented

#### 1. Divide and Conquer

* Merge Sort
* Efficient for large datasets
* Time Complexity: O(n log n)

#### 2. Sorting Comparison

* Merge Sort
* Quick Sort
* Python Built-in Sort
* Performance comparison using execution time and graphs

#### 3. Greedy Algorithms

* Fractional Knapsack
* Activity Selection
* Fast and efficient but may not always give optimal results

#### 4. Dynamic Programming

* 0/1 Knapsack
* Fibonacci (Recursion vs DP)
* Optimized solution using stored results

#### 5. Travelling Salesman Problem (TSP)

* Implemented using brute force
* Demonstrates exponential time complexity

---

### 📊 Performance Analysis

* Execution time measured for different input sizes
* Graphs plotted using matplotlib
* Comparison of algorithm efficiency

---

### ▶️ How to Run

1. Open the notebook in Google Colab
2. Run all cells step-by-step
3. Observe outputs and graphs

---

### 📁 Repository Structure

* `algo_strategies_notebook.ipynb` → Main notebook
* `README.md` → Project documentation

---

### 📌 Conclusion

* Divide and Conquer is efficient for sorting problems
* Greedy algorithms are fast but not always optimal
* Dynamic Programming ensures optimal solutions but uses more memory
* TSP shows exponential growth, making it impractical for large inputs
